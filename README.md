# 🏦 Sistema Bancario Web

Aplicación web para la gestión de clientes, cuentas y transacciones bancarias.

---

## 📌 Tecnologías

* Backend: Spring Boot + Hibernate
* Frontend: Angular
* Base de datos: PostgreSQL

---

## 🚀 Funcionalidades

* Gestión de clientes
* Administración de cuentas
* Registro de transacciones
* Autenticación de usuarios

---

## 📦 Instalación

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
ng serve
```

---

## ⚙️ Configuración

Editar el archivo `application.properties`:

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/banco_db
spring.datasource.username=postgres
spring.datasource.password=tu_password
```

---

## ▶️ Uso

Abrir en el navegador:

```
http://localhost:4200
```
